# PRD

## 서비스 한 줄 요약
AI·디지털·업무 적용 역량을 진단하여 수강자 유형을 분류하고, 맞춤 교육 과정과 상담 메시지를 자동 생성하는 사전 진단 서비스.

## 문제 정의
| 문제 | 설명 |
|---|---|
| 교육 대상 불명확 | 모든 신청자가 동일한 교육 대상이 아니므로 선별이 필요하다. |
| 수준 편차 | AI 경험, 디지털 도구 활용, 업무 적용 목적이 서로 다르다. |
| 추천 기준 부재 | 교육 추천이 상담자 경험에 의존한다. |
| 상담 비효율 | 강점, 보완점, 추천 이유를 매번 수동 작성해야 한다. |

## 목표 사용자
- 예비 수강자
- 강사 및 교육 상담자
- 교육기관 운영자
- 사내 AI 교육 담당자

## 핵심 가치
- 사전 진단으로 교육 적합도를 판단한다.
- 강사의 교육 30선 중 적합한 교육을 1~3순위로 추천한다.
- 비대상자 또는 보류형 수강자를 무리하게 교육 대상으로 만들지 않는다.
- 상담자가 바로 사용할 수 있는 안내 메시지를 생성한다.

## MVP 범위
- 진단 시작 화면
- 기본 정보 입력
- 20개 진단 문항
- 점수 계산
- 자동 보류 판단
- 수강자 유형 분류
- 교육 추천
- 결과 리포트
- 관리자 대시보드

## 제외 범위
- 결제
- LMS 전체 기능
- 이메일/문자 자동 발송
- 외부 CRM 연동
- 고급 머신러닝 추천 모델
- 개인정보 기반 회원 관리

## 성공 지표
| 지표 | 목표 |
|---|---:|
| 진단 완료율 | 70% 이상 |
| 평균 진단 시간 | 10분 이내 |
| 추천 결과 표시율 | 100% |
| AI 실패 시 기본 결과 제공 | 100% |
| 관리자 통계 제공 | 1차 MVP 포함 |
