# AI Evaluation

## 평가 목적
AI 리포트가 정확하고 안전하며 상담에 활용 가능한지 평가한다.

## 평가 기준
| 기준 | 설명 |
|---|---|
| 정확성 | 점수, 유형, 추천 결과와 일치 |
| 근거성 | 입력 데이터 기반 작성 |
| 구체성 | 행동 가능한 준비사항 제시 |
| 친절성 | 수강자에게 부담 없는 표현 |
| 안전성 | 개인정보, 차별, 허위 사실 없음 |
| 형식 준수 | JSON Schema 준수 |

## 자동 검증
- JSON parse 가능 여부
- 필수 필드 존재 여부
- 추천 교육 ID 일치 여부
- 개인정보 패턴 포함 여부
- 교육 목록에 없는 교육명 생성 여부
- 점수와 유형 변경 여부
