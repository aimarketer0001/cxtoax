# Project Rules

## 프로젝트 개요
수강 전 역량 진단 및 맞춤 교육 추천 서비스는 교육 신청 전 수강자의 역량과 목적을 진단하여 적합한 교육을 추천하는 서비스다.

## 코딩 규칙
- TypeScript 기반 구현을 권장한다.
- 점수 계산, 유형 분류, 추천 로직은 UI에서 분리한다.
- AI 호출은 서버 측에서만 수행한다.
- 환경변수는 `.env.example`에 키 이름만 둔다.
- 문서에 없는 기능을 MVP에 임의로 추가하지 않는다.

## 문서 규칙
- docs 폴더의 문서를 기준으로 구현한다.
- 불명확한 요구사항은 `가정`으로 분리한다.
- 개인정보, API Key, 비밀번호는 문서와 코드에 포함하지 않는다.

## 보안 정책
- 이름, 전화번호, 이메일, 주소 등 직접 식별 정보는 기본 입력 항목에서 제외한다.
- 자유 입력값은 저장 전 또는 AI 전달 전 마스킹한다.
- 관리자 화면은 인증된 사용자만 접근한다.

## 완료 기준
- 20개 문항 응답, 점수 계산, 유형 분류, 교육 추천, 결과 리포트, 관리자 통계가 동작한다.
