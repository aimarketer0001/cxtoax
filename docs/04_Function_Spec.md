# Function Spec

## 기능 목록
| 기능 ID | 기능명 | 설명 | 입력 | 처리 | 출력 | 우선순위 |
|---|---|---|---|---|---|---|
| F-001 | 진단 시작 | 목적/개인정보 안내 표시 | 없음 | 안내 렌더링 | 시작 CTA | Must |
| F-002 | 기본 정보 입력 | 직무/목적/관심 수집 | 선택값 | validation | profile | Must |
| F-003 | 문항 응답 | 20개 문항 응답 수집 | answers | 진행률/검증 | answer payload | Must |
| F-004 | 점수 계산 | 영역별 점수와 총점 산출 | answers | score rule | scores | Must |
| F-005 | 보류 판단 | 자동 보류 조건 적용 | scores/profile | rule check | hold reason | Must |
| F-006 | 유형 분류 | 수강자 유형 산출 | scores | classify | learner type | Must |
| F-007 | 교육 추천 | 30개 과정 중 1~3순위 | profile/scores | match score | recommendations | Must |
| F-008 | 리포트 생성 | 강점/보완점/준비사항 | result | template/AI | report | Must |
| F-009 | 관리자 통계 | 유형/추천 교육 통계 | date filter | aggregate | dashboard | Should |

## 점수 체계
| 평가 영역 | 문항 | 배점 |
|---|---|---:|
| AI 이해도 | 1~4번 | 20 |
| 디지털 활용 역량 | 5~7번 | 15 |
| 업무 적용 역량 | 8~12번 | 25 |
| 기획 및 문제정의 역량 | 13~16번 | 20 |
| 학습 의지 및 실행력 | 17~20번 | 20 |

## 유형 분류
| 유형 | 조건 |
|---|---|
| 입문 체험형 | 총점 40~59 또는 AI 이해도 10점 이하 |
| 업무 적용형 | 총점 60~84 및 업무 적용 역량 15점 이상 |
| 기획 확장형 | 총점 75점 이상 및 기획 역량 14점 이상 |
| 전문가 성장형 | 총점 85점 이상 및 실행력 16점 이상 |
| 비적합 또는 보류형 | 총점 39점 이하 또는 자동 보류 조건 해당 |

## 자동 보류 조건
- 학습 의지 및 실행력 10점 미만
- 디지털 활용 역량 6점 미만
- 단순 호기심 + 결과물 목표 없음
- 제공 교육 범위를 벗어난 요구
- 실습/피드백 참여 의사 낮음
