# 첨부 서비스 정의 분석 요약

## 서비스 한 줄 요약
수강자의 AI·디지털·업무 적용 역량을 사전에 진단하고, 30개 교육 과정 중 적합한 교육을 추천하는 맞춤형 교육 추천 서비스.

## 핵심 사용 시나리오
1. 수강자가 진단을 시작한다.
2. 기본 정보와 20개 문항에 응답한다.
3. 시스템이 영역별 점수와 총점을 계산한다.
4. 수강자 유형을 분류한다.
5. 1~3순위 교육을 추천한다.
6. AI가 리포트와 상담 메시지를 생성한다.
7. 관리자는 통계를 보고 교육 운영 방향을 조정한다.

## 주요 화면
- 시작 화면
- 기본 정보 입력
- 진단 질문
- 결과 요약
- 추천 교육
- 상세 리포트
- 관리자 로그인
- 관리자 대시보드
- 응답자 목록
- 응답 상세
- 교육 과정 관리

## 불명확한 요구사항
- 공식 교육 목표와 세부 커리큘럼
- 공식 난이도
- 상담 신청 방식
- 결과 저장 기간
- 관리자 계정 운영 방식
- 실제 수강 신청/결제 연계 여부

## 개발 시 주의할 점
- 추천 결정은 규칙 기반으로 구현한다.
- AI는 추천 결과를 변경하지 않도록 제한한다.
- 자유 입력값은 개인정보 마스킹을 적용한다.
- 보류형 사용자에게도 대안과 재진단 경로를 제공한다.
